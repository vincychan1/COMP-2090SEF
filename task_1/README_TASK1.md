# OOP Supermarket Checkout System

An object-oriented supermarket checkout simulation system implemented in Python, supporting product management, shopping cart operations, discount strategies, and receipt saving functionality.

## Features

- **Product Management**: Add, delete, modify product information, data saved in JSON files
- **Shopping Cart Operations**: Add items, modify quantities, remove items, clear cart
- **Discount Strategies**: Support no discount, member discount, full reduction and more
- **Receipt Generation**: Generate detailed receipts after checkout, save as JSON or text files
- **Data Persistence**: All product data automatically saved, supports loading after program restart
